- 📄 Resume Upload & Analysis
- 🎯 Job Description Matching
- 🎤 Adaptive AI Interviews
- 🧠 Skill Verification
- 👥 Recruiter Dashboard
- 📊 Hiring Readiness Score
- 🔐 Secure Authentication

- Google Gemini AI

- JWT Authentication

## 🚀 Run Locally

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8000
```
